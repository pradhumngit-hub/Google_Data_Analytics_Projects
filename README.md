# Financial Dashboard Automation

## Overview
This project automates the process of financial data collection, cleaning, and visualization using Google Drive, Outlook, Python, and Power BI. It helps reduce manual work, errors, and operational costs by integrating various tools and techniques.

## Problem
The team was receiving 25+ files daily containing financial data from different locations. Manually downloading, combining, and cleaning these files was time-consuming and error-prone.

## Solution
- Used Outlook rules and Power Automate to filter emails and download attachments.
- Integrated Google Drive API to store and manage files automatically.
- Processed data using Python (pandas) for cleaning and transformation.
- Built interactive dashboards in Power BI to provide insights like income trends, payment delays, and credit utilization.

## Key Features
✔ Automated file processing  
✔ Data cleaning and error reduction  
✔ Interactive visualizations for better decision-making  
✔ Cost optimization by minimizing manual effort

## Tools & Technologies
- Power BI  
- Python (pandas, matplotlib, etc.)  
- Google Drive API  
- Outlook & Power Automate

## How to Use
1. Clone the repository.
2. Setup your API keys and credentials.
3. Run Python scripts to fetch and clean data.
4. Open the Power BI dashboard to explore insights.
